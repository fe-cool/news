MDH • 前端情报

2021.04.16

  • webpack SVG Import as React Component 的方式并不好，因为 bundle 到一起，没有用到的也会加载，所以会影响加载性能和内存占用，那么推荐的方式是啥？import 后返回一个 url 可能比较合适，比如用 webpack 的 url-loader

  • esbuild esmon，类似 nodemon 但基于 esbuild，支持 ts 等，试过还挺好用

  • vsc theme VSCode 里可以用 Github 主题了

  • babel Babel 新增 Feature Timeline，把之前 changelog 里的重要功能摘了出来，一目了然

  • react 可以用 React 做视频的 remotion 发布 2.0，支持音频，使用 esbuild-loader 代替 babel-loader，<Gif> 组件等

  • tools Fusuma 一款文档演示工具，可根据一定的标识将文档分割成不同页面，以 slide 的形式进行分享。以 ppt 的形式演示文档，能更好的突出重点。在 lerna 的管理下，项目结构很清晰，适合直接部署或者二次开发，比如基于它扩展出手机端的翻页笔、标注工具之类的。甚至可以集成在博客里，直接启动 slide

  • tools rive2 (rive1 === flare) - 基于 Flutter 开发的网页端动画制作工具(站点支持 PWA) 提供 Web(基于 wasm)、Flutter、Cpp、iOS、Android 五种运行时，集成起来挺方便的。同时也体现了 Flutter Web 的表现能力，使用体验还不错。

  • rust libp2p-rs rust实现的libp2p

  • rust rbatis 用rust实现ORM框架

  • rust rust 9.2K 跨平台 GUI 仓库 iced

  • rust swc rust编写的超快速的typescript/ javascript编译器,性能秒杀tsc

  • vue vue-cli维护者蒋豪群，给大家讲如何学习新的技术（以vue3个vite为例）
